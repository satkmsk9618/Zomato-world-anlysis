## Project Overview
This project features an interactive **Power BI dashboard** analyzing Zomato restaurant data to uncover key insights into the global restaurant market. It focuses on **geographical distribution, restaurant performance, cost trends, and customer ratings** across international regions.  

The goal is to provide a clear, actionable overview for stakeholders interested in the restaurant industry landscape.

### 1. World Analysis
- **Global Footprint Map:** Visualizes restaurant density across continents (Europe, Asia, Africa, North America, South America, Oceania).  
- **High-Value Restaurants:** Displays top 5 restaurants by average cost (e.g., Sushi Masa, Satoo Hotel Shangri-La).  
- **Key Filters:** Filter by Country, City, Rating Color, Table Booking, Online Delivery.

### 2. Performance Analysis
- **Brand Dominance Treemap:** Highlights prominent restaurant chains (Cafe Coffee Day, Domino's Pizza, Subway, McDonald's, Pizza Hut, Baskin Robbins).  
- **Currency & Cost Context:** Analysis in Botswana Pula (BWP) with Price Range filter.  
- **Location Spotlight:** Specific location analysis (e.g., Bakers & More, Faridabad).
